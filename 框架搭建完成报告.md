- ✅ **KISS原则** - 代码简洁直观，避免过度复杂

## 🚀 快速启动

### Windows环境
```bash
# 双击运行启动脚本
start-dev.bat
```

### Linux/Mac环境
```bash
# 给脚本执行权限并运行
chmod +x start-dev.sh
./start-dev.sh
```

### 手动启动

#### 后端启动
```bash
cd yxly-backend
mvn spring-boot:run
```

#### 前端启动
```bash
cd yxly-frontend
npm install
npm run dev
```
