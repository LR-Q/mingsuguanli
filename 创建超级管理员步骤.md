# 创建超级管理员账号

## 方法一：使用MySQL客户端（推荐）

### 步骤1：打开MySQL客户端
```bash
mysql -u root -p123456
```

### 步骤2：复制并执行以下SQL

```sql
USE yxly_dev;

INSERT INTO sys_user (
    username, 
    password, 
    real_name, 
    email, 
    phone, 
    role_id, 
    status, 
    email_verified, 
    phone_verified, 
    balance, 
    create_time, 
    update_time, 
    deleted
) 
SELECT 
    'superadmin',
    '$2a$10$5Z8FvBLOF4KqVYj8nHYOLu0R3jVx7xqK0h.JF2nYpQGRiN9QE.W5K',
    '超级管理员',
    'super@admin.com',
    '13900000000',
    id,
    1,
    1,
    1,
    0.00,
    NOW(),
    NOW(),
    0
FROM sys_role 
WHERE role_code = 'SUPER_ADMIN';
```

### 步骤3：验证创建结果
```sql
SELECT 
    u.id,
    u.username,
    u.real_name,
    r.role_name,
    r.role_code,
    u.status
FROM sys_user u
LEFT JOIN sys_role r ON u.role_id = r.id
WHERE u.username IN ('superadmin', 'admin') 
AND u.deleted = 0
ORDER BY u.username;
```

应该看到两个账号：
- admin（你原有的账号，保持不变）
- superadmin（新创建的超级管理员）
